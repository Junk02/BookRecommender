Semantic search powered by LLM embeddings
This project is an intelligent web application designed to search and recommend books based on a free‑form text description provided by the user. Instead of relying on keyword matching, the system uses LLM‑based embeddings to understand the semantic meaning of the query and find the most relevant books in the database.
The goal of the project is to demonstrate how modern AI technologies — large language models, vector embeddings, and semantic similarity search — can be applied to improve the process of discovering literature.
